# AnimateX Architecture

## Overview
AnimateX is built using a microservices architecture, with separate services for:
- Web Interface
- API Gateway
- AI Processing
- Blockchain Integration
- Storage Management

## System Components
1. Frontend (React + TypeScript)
2. Backend API (Node.js + Express)
3. AI Service (Python + TensorFlow)
4. Blockchain Service (Solidity + Web3)
5. Storage Service (IPFS + Arweave)
